# 图书管理系统 (Library Management System)

## 系统介绍
图书管理系统是一个基于Qt框架开发的桌面应用程序，用于图书馆的日常管理工作，包括图书管理、读者管理、借阅管理等功能。系统采用SQLite数据库进行数据存储，提供了友好的用户界面和完整的功能模块。

## 功能特点

### 1. 用户认证系统
- **用户注册**：支持新用户注册
- **用户登录**：支持已注册用户登录
- **权限管理**：默认管理员账号和普通用户账号

### 2. 图书管理
- **图书查询**：支持按书名、作者、出版社、ISBN等条件查询
- **图书添加**：支持添加新图书信息
- **图书修改**：支持修改现有图书信息
- **图书删除**：支持删除图书信息
- **图书分类**：支持图书分类管理

### 3. 读者管理
- **读者查询**：支持按姓名、读者编号、身份证号等条件查询
- **读者添加**：支持添加新读者信息
- **读者修改**：支持修改现有读者信息
- **读者删除**：支持删除读者信息

### 4. 借阅管理
- **图书借阅**：支持读者借阅图书
- **图书归还**：支持读者归还图书
- **借阅记录**：支持查询借阅历史记录
- **逾期提醒**：支持显示逾期未还的图书

## 技术栈

- **开发框架**：Qt 6.9.2
- **编程语言**：C++
- **数据库**：SQLite
- **UI设计**：Qt Designer
- **开发环境**：Qt Creator

## 安装方法

### 系统要求
- Windows 10/11
- Qt 6.9.2或更高版本
- MinGW 64位编译器

### 编译步骤

1. **克隆或下载项目**
   ```bash
   git clone https://github.com/yourusername/LibraryManagementSystem.git
   cd LibraryManagementSystem
   ```

2. **使用Qt Creator打开项目**
   - 打开Qt Creator
   - 点击"打开项目"
   - 选择项目目录中的`LibraryManagementSystem.pro`文件

3. **配置项目**
   - 选择合适的构建套件（推荐Qt 6.9.2 + MinGW 64位）
   - 点击"配置项目"

4. **编译项目**
   - 点击"构建"菜单中的"构建项目"
   - 等待编译完成

5. **运行项目**
   - 点击"运行"菜单中的"运行"选项
   - 或直接点击运行按钮

## 使用说明

### 1. 登录系统

- 启动应用程序后，首先进入登录界面
- 默认管理员账号：`admin`，密码：`admin123`
- 点击"注册"按钮可以创建新用户账号
- 登录成功后进入系统主界面

### 2. 图书管理

- 在主界面左侧导航栏中选择"图书管理"
- 可以查看所有图书列表
- 点击"添加"按钮添加新图书
- 选择图书后点击"修改"或"删除"按钮进行相应操作
- 使用搜索框可以快速查询图书

### 3. 读者管理

- 在主界面左侧导航栏中选择"读者管理"
- 可以查看所有读者列表
- 点击"添加"按钮添加新读者
- 选择读者后点击"修改"或"删除"按钮进行相应操作
- 使用搜索框可以快速查询读者

### 4. 借阅管理

- 在主界面左侧导航栏中选择"借阅管理"
- 可以查看所有借阅记录
- 点击"借阅"按钮为读者办理借阅手续
- 选择借阅记录后点击"归还"按钮办理归还手续
- 使用筛选功能可以查看逾期未还的图书

## 数据库说明

系统使用SQLite数据库，数据库文件会在第一次运行时自动创建。数据库包含以下主要表：

- **books**：存储图书信息
- **readers**：存储读者信息
- **borrow_records**：存储借阅记录
- **users**：存储用户账号信息

## 开发注意事项

1. **数据库初始化**：系统会在第一次运行时自动初始化数据库和创建默认管理员账号
2. **权限管理**：当前版本实现了基本的用户认证，但权限管理功能可以进一步扩展
3. **密码安全**：当前版本密码以明文存储，生产环境中建议使用加密存储
4. **数据备份**：建议定期备份SQLite数据库文件
